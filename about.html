    About Checkers.io

    Checkers.io is a simply made checkers browser game which demonstrates the use of HTML, CSS, and JS in the making of a website. A later application will be made utilizing this format for the purpose of 2-player interaction and gameplay.

    Our use of the name and the game is for non-profit educational use only. No part of this code or program should be used outside of that definition.
